# CUK밥 변경 로그

## 버전 2.2.0

### 추가됨
- 공지사항 시스템: 앱의 최신 소식을 확인할 수 있도록 화면 상단에 새로운 공지사항 버튼(확성기 아이콘)을 추가했습니다.
- 스마트 알림: 새로운 공지사항이 게시되거나 메뉴 정보가 업데이트될 때마다 알림을 받을 수 있습니다.
- 알림 설정: 설정에 '공지사항 알림' 토글을 추가하여 받고 싶은 알림을 직접 제어할 수 있습니다.
- 다국어 지원 개선: 이제 공지사항과 업데이트 로그가 사용자가 선택한 언어(한국어, 일본어, 중국어, 영어)로 자동 표시됩니다.

### 변경됨
- 더 스마트해진 첫 실행: 앱 설치 후 튜토리얼을 마칠 때까지 새로운 알림이 대기하여 더욱 깔끔한 첫인상을 제공합니다.

### 개선됨
- 백그라운드 확인: 배터리 효율을 높이고 안정성을 강화하기 위해 백그라운드 업데이트 확인 방식을 개선했습니다.
- 다국어 지원 강화: 모든 새로운 설정 항목과 알림 메시지를 한국어, 일본어, 중국어 사용자에 맞춰 현지화했습니다.

## 버전 2.1.0

새로운 기능
- 새로운 버전이 출시되었을 때 알림을 표시하는 인앱 업데이트 확인 기능 추가
- 신규 사용자가 앱을 쉽게 이해할 수 있도록 온보딩 튜토리얼 시스템 추가
- 고정한 메뉴에 대한 알림 기능 추가
- 특정 날짜의 식단을 빠르게 확인할 수 있는 스마트 캘린더 추가
- 이전에 제공된 식단을 확인할 수 있는 메뉴 기록 기능 추가
- 사용자 맞춤 인터페이스를 위한 강조 색상 설정 기능 추가
- 접기/펼치기 가능한 메뉴 섹션 추가 (상태 유지)

계정 및 보안
- 앱 내에서 계정을 삭제할 수 있는 기능 추가
- 보안 강화를 위해 Google 로그인 시스템을 최신 버전으로 업데이트

신고 기능 개선
- 처음 신고를 제출할 때 경고 메시지 표시 기능 추가
- 신고 제출 후 성공 또는 실패 여부를 알려주는 확인 팝업 추가
- 신고가 성공적으로 제출되면 생성된 GitHub 이슈 링크 표시

투명성 및 오픈소스

- 메뉴 파싱에 사용되는 GitHub 저장소 링크 추가
- 앱 내 개인정보 처리방침 링크 추가
- 개인정보 처리방침 업데이트

수정
- 위젯이 정상적으로 로드되지 않던 문제 수정
- “새로운 기능(What’s New)” 페이지가 기본적으로 영어로 표시되던 문제 수정

개선
- 전반적인 안정성과 신뢰성 개선
- 여러 화면에서 UI 개선
- 기타 사소한 개선 및 버그 수정
