## 버전 2.1.0

새로운 기능
- 새로운 버전이 출시되었을 때 알림을 표시하는 인앱 업데이트 확인 기능 추가
- 신규 사용자가 앱을 쉽게 이해할 수 있도록 온보딩 튜토리얼 시스템 추가
- 고정한 메뉴에 대한 알림 기능 추가
- 특정 날짜의 식단을 빠르게 확인할 수 있는 스마트 캘린더 추가
- 이전에 제공된 식단을 확인할 수 있는 메뉴 기록 기능 추가
- 사용자 맞춤 인터페이스를 위한 강조 색상 설정 기능 추가
- 접기/펼치기 가능한 메뉴 섹션 추가 (상태 유지)

계정 및 보안
- 앱 내에서 계정을 삭제할 수 있는 기능 추가
- 보안 강화를 위해 Google 로그인 시스템을 최신 버전으로 업데이트

신고 기능 개선
- 처음 신고를 제출할 때 경고 메시지 표시 기능 추가
- 신고 제출 후 성공 또는 실패 여부를 알려주는 확인 팝업 추가
- 신고가 성공적으로 제출되면 생성된 GitHub 이슈 링크 표시

투명성 및 오픈소스
- 오픈소스 라이선스 추가
- 메뉴 파싱에 사용되는 GitHub 저장소 링크 추가
- 앱 내 개인정보 처리방침 링크 추가
- 개인정보 처리방침 업데이트

수정
- 위젯이 정상적으로 로드되지 않던 문제 수정
- “새로운 기능(What’s New)” 페이지가 기본적으로 영어로 표시되던 문제 수정

개선
- 전반적인 안정성과 신뢰성 개선
- 여러 화면에서 UI 개선
- 기타 사소한 개선 및 버그 수정
