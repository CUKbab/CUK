## 버전 2.1.0

### 추가

* 앱 내 업데이트 확인 기능 추가
* 신규 사용자를 위한 온보딩 튜토리얼 시스템 추가
* 고정한 메뉴에 대한 알림 기능 추가
* 원하는 날짜의 메뉴를 빠르게 확인할 수 있는 스마트 캘린더 추가
* 이전 메뉴를 확인할 수 있는 메뉴 기록 시스템 추가
* 강조 색상(Accent Color) 설정 기능 추가
* 개인정보 처리방침 링크 추가
* 메뉴 파싱에 사용되는 GitHub 저장소 링크 추가
* 오픈소스 라이선스 추가
* 계정 삭제 기능 추가

### 변경

* 더 안전한 인증을 위해 새로운 Google Sign-In 버전으로 마이그레이션

### 수정

* 위젯이 로드되지 않는 문제 수정
* “새로운 기능” 페이지가 기본적으로 영어로 표시되던 문제 수정

### 개선

* 기타 사소한 개선 및 버그 수정
