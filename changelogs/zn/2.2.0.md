# 版本 2.2.0

## 新增
- 公告系统：在屏幕顶部新增了公告按钮（扩音器图标），让您随时了解应用的最新动态。
- 智能通知：每当发布新公告或菜单信息更新时，您现在都可以收到通知。
- 通知控制：在设置中新增了“公告通知”开关，让您完全掌控接收哪些提醒。
- 更好的本地化：公告和更新日志现在会根据您选择的语言（韩语、日语、中文或英语）自动显示。

## 变更
- 更智能的首次启动：新通知将等待您完成新手引导教程后再出现，确保更清爽的首次使用体验。

## 改进
- 后台轮询：改进了后台检查更新的方式，使其更加省电且可靠。
- 多语言支持：为韩语、日语和中文用户本地化了所有新设置和通知消息。
