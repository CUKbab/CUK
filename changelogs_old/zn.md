# CUK밥 更新日志

## 版本 2.2.0

### 新增
- 公告系统：在屏幕顶部新增了公告按钮（扩音器图标），让您随时了解应用的最新动态。
- 智能通知：每当发布新公告或菜单信息更新时，您现在都可以收到通知。
- 通知控制：在设置中新增了“公告通知”开关，让您完全掌控接收哪些提醒。
- 更好的本地化：公告和更新日志现在会根据您选择的语言（韩语、日语、中文或英语）自动显示。

### 变更
- 更智能的首次启动：新通知将等待您完成新手引导教程后再出现，确保更清爽的首次使用体验。

### 改进
- 后台轮询：改进了后台检查更新的方式，使其更加省电且可靠。
- 多语言支持：为韩语、日语和中文用户本地化了所有新设置和通知消息。

## 版本 2.1.0

新功能
- 新增应用内更新检测功能，在有新版本时通知用户
- 新增新手引导教程系统，帮助新用户快速了解应用
- 新增已固定菜单的通知功能
- 新增智能日历，可快速跳转到指定日期查看菜单
- 新增菜单历史记录功能，可浏览过去提供的菜单
- 新增主题强调色自定义功能，使界面更加个性化
- 新增可折叠菜单分区，并会记住展开状态

账户与安全
- 新增应用内账户删除功能
- 升级到更新版本的 Google 登录以提高认证安全性

反馈功能改进
- 首次提交报告前新增警告提示
- 提交报告后新增成功或失败确认弹窗
- 报告成功提交后会显示对应 GitHub Issue 的链接

透明度与开源
- 新增开源许可证
- 新增用于菜单解析的 GitHub 仓库链接
- 在应用内新增隐私政策链接
- 更新隐私政策

修复
- 修复小组件无法正常加载的问题
- 修复 “What’s New” 页面默认显示为英文的问题

改进
- 提升整体稳定性与可靠性
- 对多个界面进行了细节 UI 优化
- 其他若干小幅改进与错误修复
