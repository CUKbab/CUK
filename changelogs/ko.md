# CUK밥 변경 로그

## 버전 2.0.0
- 앱을 완전히 새로 제작
- Google Material 3 디자인을 적용한 UI 전면 개편
- 가로 모드 및 태블릿 지원 개선
- 새로운 학식 메뉴 형식에 맞게 구조 개선
- 건의사항 및 일반 버그 제보 스팸 방지를 위한 로그인 시스템 추가
- 메뉴 오류 제보는 로그인 없이 가능
- 앱 내 글자 크기 조절 기능 추가
- 앱 내 언어 변경 기능 추가 (한국어, 영어, 일본어, 중국어 - zh-rCN)
- 위젯 시스템 개선 및 재설계
- 성능 개선 및 각종 버그 수정
