## Version 2.1.0

New Features
- Added in-app update checker to notify users when a new version is available
- Added onboarding tutorial system to help new users quickly learn the app
- Added notifications for pinned meals
- Added smart calendar for quickly jumping to specific dates to check menus
- Added menu history system to browse previously served menus
- Added accent color customization for a more personalized interface
- Added collapsible menu sections with persistent state

Account & Security
- Added account deletion feature directly inside the app
- Migrated to a newer version of Google Sign-In for improved authentication security

Reporting Improvements
- Added a warning message before submitting reports for the first time
- Added confirmation popup after submitting a report indicating success or failure
- Successful reports now show a link to the created GitHub issue

Transparency & Open Source
- Added open source license
- Added link to the GitHub repository used for menu parsing
- Added privacy policy link inside the app
- Updated privacy policy

Fixes
- Fixed widgets not loading properly
- Fixed “What’s New” page defaulting to English

Improvements
- Improved overall stability and reliability
- Minor UI polish across several screens
- Various minor improvements and bug fixes
