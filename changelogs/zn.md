## 版本 2.1.0

新功能
- 新增应用内更新检测功能，在有新版本时通知用户
- 新增新手引导教程系统，帮助新用户快速了解应用
- 新增已固定菜单的通知功能
- 新增智能日历，可快速跳转到指定日期查看菜单
- 新增菜单历史记录功能，可浏览过去提供的菜单
- 新增主题强调色自定义功能，使界面更加个性化
- 新增可折叠菜单分区，并会记住展开状态

账户与安全
- 新增应用内账户删除功能
- 升级到更新版本的 Google 登录以提高认证安全性

反馈功能改进
- 首次提交报告前新增警告提示
- 提交报告后新增成功或失败确认弹窗
- 报告成功提交后会显示对应 GitHub Issue 的链接

透明度与开源
- 新增开源许可证
- 新增用于菜单解析的 GitHub 仓库链接
- 在应用内新增隐私政策链接
- 更新隐私政策

修复
- 修复小组件无法正常加载的问题
- 修复 “What’s New” 页面默认显示为英文的问题

改进
- 提升整体稳定性与可靠性
- 对多个界面进行了细节 UI 优化
- 其他若干小幅改进与错误修复
